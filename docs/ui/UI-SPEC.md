# Mission Control UI — Specification Index

Full UI/UX specification lives in the program prompt (UI-ARCH, UI-HITL, UI-FLT, etc.).

**Normative requirements for this monorepo build** are tracked as `UI-*` IDs in:

- `docs/ui/AUDIT.md` — current state  
- `docs/ui/CONTRACT.md` — Host Interface binding  
- `gates/UI-GATE-*.md` — phase gates  
- `docs/ui/CONFORMANCE.md` — final pass/fail table (GATE-5)

Aesthetic thesis: **Control Hub / Cherenkov neon ops console** with Mission Spine as signature element.
